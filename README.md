### Instruction for running the project

1. Clone the repo
   ```sh
   git clone https://github.com/sarthakg04/Container-Tracking-Dashboard.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Change directory to dyteapp
   ```sh
   cd portcast-dash
   ```
4. Start React Server
   ```sh
   npm run serve
   ```
